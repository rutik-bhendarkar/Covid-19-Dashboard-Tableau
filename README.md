# 📊 COVID-19 Dashboard (Tableau)

## 🔍 Project Overview
This project is an interactive COVID-19 dashboard built using Tableau to analyze global pandemic trends.

## 📊 Features
- Global map showing country-wise cases
- Confirmed vs Death case comparison
- Time-series trend analysis
- Top affected countries
- Interactive filters

## 🛠️ Tools Used
- Tableau
- Data Visualization
- Data Analysis

## 📈 Insights
- Rapid growth after March 2020
- US, Brazil, India among top affected
- Variation in death rates across countries
